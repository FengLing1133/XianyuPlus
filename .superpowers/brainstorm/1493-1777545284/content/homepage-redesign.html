🎨 设计要点总结

新设计的核心特征

  • 森林绿主色调 #2b9939 替代原蓝色系
  • 全圆角胶囊形状 按钮、搜索框、分类标签
  • Emoji 图标体系 导航栏、分类、商品占位图
  • 马卡龙色占位 无图商品显示纯色+Emoji
  • Hero Banner 浅绿渐变圆角横幅
  • 柔和阴影悬浮 卡片 hover 上浮+阴影加深
  • 红色醒目价格 + 灰色中划线原价

需要确认的方向

  • 是否保留 Element Plus 组件库（可大幅减少代码量）
  • 还是改为纯 CSS 实现（更灵活但工作量大）
  • 管理后台是否也要同步改造
  • 字体方案偏好（系统默认 vs 引入 Web 字体）
