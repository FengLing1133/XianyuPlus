Vue 3 + Spring Boot 多模块 + MySQL + Redis

系统架构图

service

↓
MySQL 8.0

模块职责

common实体类、DTO、枚举、公共异常、工具类
frameworkSpring Security + JWT、Redis 配置、WebSocket 配置、全局异常处理、跨域配置
service核心业务：用户、商品、收藏、订单、文件上传
chatWebSocket 即时通讯，Redis 消息队列
admin管理后台接口：用户管理、商品审核、数据统计

技术选型总结

前端Vue 3 + Element Plus + Pinia + Vue Router + Axios
后端Spring Boot 2.7 + MyBatis-Plus + Spring Security + JWT
数据库MySQL 8.0 + Redis
聊天WebSocket（Spring WebSocket） + Redis 消息队列
图片存储本地文件系统（预留 OSS 接口）
构建工具Vite（前端） + Maven（后端）
